# CppRefl
CppRefl is a reflection compiler for C++, which allows a program to examine its own data structures at runtime. 

## Project Structure
* CppRefl - A static library that contains all reflection data structures, markup, and generation code.
* CppReflGen - An application that can be used to generate and store reflection data outside of the normal application lifetime.
* CppReflTest - Testing grounds for CppRefl.
